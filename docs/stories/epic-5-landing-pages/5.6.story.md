# Story 5.6: /blog (Blog Listing Page)

**Status:** Done

**Complexity:** Medium (3 points)

## Description

Create Blog Listing page showcasing articles, tutorials, and updates about Safety Vote, voting best practices, and election management.

## Acceptance Criteria

- [x] AC1: Blog post grid/list layout with featured post
- [x] AC2: Post cards showing title, excerpt, author, date, reading time
- [x] AC3: Category/tag filtering
- [x] AC4: Search functionality for posts
- [x] AC5: Pagination or infinite scroll
- [x] AC6: CTA to subscribe to newsletter

## Scope

### IN
- Hero section with blog intro
- Featured post showcase (large card)
- Blog post grid:
  - Post title, excerpt, author, date, reading time
  - Category/topic tags
  - Featured image thumbnail
  - Link to full post
- Filtering:
  - By category (Security, Features, Tutorials, Updates)
  - By tag
- Search functionality (client-side or API)
- Pagination (10 posts per page)
- Newsletter signup CTA
- Header/footer integration

### OUT
- Individual blog post pages (separate epic)
- Comment system
- Related posts recommendations
- Author profiles (separate page)

## Dependencies

- Header/footer components
- Design system tokens
- Blog data/CMS integration (if available)
- Search library (if client-side search)

## Technical Notes

- Next.js page: `/app/blog/page.tsx`
- Blog data can be static array initially (migrate to CMS later)
- Implement pagination or virtualization for performance
- Search with JavaScript `Array.filter()` initially
- Image optimization
- Metadata for SEO (canonical URLs)
