# Story 5.9: /central-de-ajuda (Help Center Page)

**Status:** Done

**Complexity:** Medium (3 points)

## Description

Create Help Center page with searchable FAQ, knowledge base articles, and support resources for users seeking self-service help.

## Acceptance Criteria

- [x] AC1: FAQ section with collapsible Q&A items
- [x] AC2: Search functionality across help articles
- [x] AC3: Help categories/topics (Getting Started, Voting, Results, etc.)
- [x] AC4: Knowledge base article cards
- [x] AC5: Contact support CTA for unsolved issues
- [x] AC6: Popular articles and recent updates

## Scope

### IN
- Hero section with search bar
- Search functionality:
  - Search across FAQs and articles
  - Display matching results
  - Highlight search terms
- FAQ section:
  - 20+ common questions
  - Organized by category
  - Collapsible Q&A format
  - Related articles links
- Help categories:
  - Getting Started
  - Creating Elections
  - Voting Guide
  - Results & Analysis
  - Account Management
  - Security & Privacy
  - Troubleshooting
- Knowledge base articles:
  - Article cards with preview
  - Reading time estimate
  - Category tags
  - Link to full article
- Popular articles widget
- Contact support section (for unanswered questions)
- Feedback section (Was this helpful?)
- Header/footer integration

### OUT
- Full knowledge base system (separate CMS)
- Video tutorials
- Live chat support

## Dependencies

- Header/footer components
- Design system tokens
- FAQ and article content (support team)
- Search library (or simple client-side search)
