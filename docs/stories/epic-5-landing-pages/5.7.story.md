# Story 5.7: /carreiras (Careers Page)

**Status:** Draft

**Complexity:** Medium (3 points)

## Description

Create Careers page highlighting company culture, open job positions, benefits, and hiring process.

## Acceptance Criteria

- [x] AC1: Company culture/values section
- [x] AC2: Benefits overview (salary, health, remote, etc.)
- [x] AC3: Open positions listing with job details
- [x] AC4: Application/contact CTA per position
- [x] AC5: Team testimonials or culture highlights
- [x] AC6: Link to external job board or careers portal

## Scope

### IN
- Hero section with "Join Our Team" message
- Company culture section:
  - Core values alignment
  - Work environment description
  - Team diversity statement
- Benefits section:
  - Competitive salary
  - Health insurance
  - Remote-first policy
  - Professional development
  - Flexible hours
- Open positions:
  - Position cards with:
    - Title, seniority level, department
    - Description/summary
    - Required skills
    - Location (remote/hybrid)
    - CTA button (Apply)
- Team testimonials (optional)
- Hiring process explanation
- Contact HR section
- Header/footer integration

### OUT
- Full careers portal (separate ATS integration)
- Video testimonials
- Detailed job descriptions (can be on external board)
- Application tracking system

## Dependencies

- Header/footer components
- Design system tokens
- Current job openings list (HR team)
- Company culture content
- Benefits information

## Technical Notes

- Next.js page: `/app/carreiras/page.tsx`
- Job positions can be static array or API-fetched
- Link to external job board (Lever, Greenhouse, etc.)
- Responsive cards for positions
- Metadata for SEO
